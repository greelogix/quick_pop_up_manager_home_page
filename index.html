üöÄ Quick Start v0.0.1
1. Add to MaterialApp: 
MaterialApp(
  navigatorObservers: [
    QuickPopupNavigatorObserver()
  ],
  // ...
)
2. Use anywhere (no BuildContext!): 
// Toast
QuickPopupManager()
  .showSuccessToast(
    message: 'Success!',
    position: PopupPosition.top,
  );

// Dialog
QuickPopupManager()
  .showDialogPopup(
    title: 'Confirm',
    message: 'Are you sure?',
    onConfirm: () {},
  );

// Bottom Sheet
QuickPopupManager()
  .showBottomSheet(
    title: 'Options',
    content: YourWidget(),
  );
